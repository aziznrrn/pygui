## Setup
1. Clone this repository
2. Make sure you have python3 and pip installed
    ```bash
        python --version
        pip --version
    ```
3. Create virtual environment
    ```bash
        python -m venv venv
    ```
4. Install requirements
    ```bash
        pip install -r requirements.txt
    ```
5. Run the code
    ```bash
        python main.py
    ```
